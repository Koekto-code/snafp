
## snafp

Some Nest.js App For Practice
Original repo from course: https://github.com/dadi2015/lesson_nestjs

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
